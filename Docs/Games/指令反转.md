---

## 做什么

屏幕显示身体指令，玩家需做**相反动作**（Simon Says 反转规则）。

| 显示指令 | 玩家应做 |
|----------|----------|
| 举右手 | **举左手**（MediaPipe 举手检测） |
| 举左手 | **举右手** |
| 向右一步 | **身体右倾**（肩线偏移） |
| 向左一步 | **身体左倾** |

- **摇头** → 跳过本回合  
- 专注力越高，指令间隔越短

---

## BCI / 动作映射

| 信号 | 作用 |
|------|------|
| 专注力 | 指令刷新间隔 |
| `RightHandRaiseEdge` | 响应「举右手」类指令（举左手） |
| `LeftHandRaiseEdge` | 响应「举左手」类指令 |
| `BodyLean` ±1 | 踏步类相反动作 |
| 摇头 | 跳过 |
