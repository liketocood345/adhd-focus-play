## 做什么

经典 **Go/No-Go** 变体：物体从上方落下。

| 物体 | 规则 |
|------|------|
| **晶石**（绿灯） | **眨眼**收集 → 得分 |
| **陨石**（红灯） | **不要眨眼**；误眨扣分 |

专注力越高，下落速度越快。**摇头**可清除场上陨石（每局限 3 次「激光」）。

训练目标：对优势反应（收集）进行抑制控制，对应冲动性与抑制缺陷。

---

## BCI 映射

| 信号 | 作用 |
|------|------|
| 专注力 | 下落速度倍率 |
| 眨眼 | 晶石 Go 反应 |
| 抑制眨眼 | 陨石 No-Go |
| 摇头 | 清除陨石（3 次） |
